This project focuses on analyzing business and financial data using SQL to simulate FP&A scenario and business analysis scenarios. 
The queries are used to calculate: 
- Performance metric (Monthly Revenue, COGS, and Gross Margin %)
- Analytics metric (Window Functions for YoY growth and 3-month rolling averages)
- Variance Analysis (Actual vs. Budget)
- Strategic Insights (Rank Top product and customer by profitability)
